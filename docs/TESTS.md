TESTS
======

The tests use mocha with chai to run.
It is already configured in solution.


## Run tests

```sh
npm run test
```

### Run unique test

```sh
npm run mocha {path to your test}
ex: npm run mocha app/tests/app.spec.js
```

### Run test on html report 

```sh
npm run test:html
```

### Run tes t on coverage mode

```sh
npm run test:cov
```
